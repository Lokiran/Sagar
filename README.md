A SQL Server database project for managing student academic records. It implements a normalized (3NF) schema with four main tables:

Department - Stores department information
Student - Contains student details with roll numbers and department assignment
Subject - Lists all subjects offered
Result - Records student marks for each subject by semester
The project includes SQL query files for creating tables, inserting data, and performing various database operations to track and analyze student performance across semesters.

Files included:

SQLQuery5.sql - Database schema creation (tables and constraints)
SQLQuery6.sql - Data insertion/queries
SQLQuery7.sql - Additional queries
SQLQuery8.sql - Supplementary query
